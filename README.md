# Smart College Canteen

A full-stack MERN application for college canteen management, featuring online ordering, real-time tracking, student/staff/admin portals, and sales analytics.

## Quick Start

### Prerequisites
- Node.js (v14+)
- MongoDB (Running on `localhost:27017`)

### Installation

1.  **Setup Backend**
    ```bash
    cd server
    npm install
    npm run dev
    ```

2.  **Setup Frontend**
    ```bash
    cd client
    npm install
    npm run dev
    ```

3.  Access the app at `http://localhost:5173`.

## Features
- **Student**: Browse Menu, Cart, Order Placement, Order History.
- **Staff**: Kitchen Dashboard for real-time order status updates.
- **Admin**: Menu Management, QR Code Generation, Sales Analytics.
